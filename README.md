# WWF & CYP: Cyber Love 520 🎮

一个浪漫的赛博风格跑酷游戏，为 WWF 和 CYP 特别定制！

## 🎯 游戏说明

- **目标**：跑完 520 米，营救 CYP 公主
- **操作**：
  - 💻 电脑：空格键跳跃 / F 键射击
  - 📱 手机：点击屏幕跳跃 / 按钮射击
  - 支持空中二段跳！

## 🚀 在线游玩

访问：https://你的用户名.github.io/仓库名/feng.html

## 📦 本地运行

### 方法一：Python 服务器（推荐）

```bash
python start_server.py
```

然后访问：http://localhost:8000/feng.html

### 方法二：Node.js

```bash
npm install -g http-server
http-server -p 8000
```

## 🎨 游戏特色

- ✨ 精美的赛博朋克风格
